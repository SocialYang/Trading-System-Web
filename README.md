# web_ctp

目标:使用web界面监控ctp期货接口的运行

当前:web界面用brython控制，通过websocket与ctp交互，ctp通过zmq与策略层(zmq_server中)通信

技术交流群  515942461
=====================

环境搭建
========
win7 x86

安装activepython2.7 x86

安装vcredist x86最新版

运行pip_install.bat

运行_TEST_.bat检查环境是否ok

试运行
======
启动_ZMQ_.bat

启动_RUN_.bat

用firefox打开ctp.html
